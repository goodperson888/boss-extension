# BOSS直聘自动投递助手 - 安装使用指南

## 🎯 功能特点

- ✅ 智能职位匹配（关键词、薪资、经验）
- ✅ 自动批量投递
- ✅ 模拟人工行为（随机延迟、滚动、鼠标悬停）
- ✅ 防止重复投递
- ✅ 投递记录管理
- ✅ 美观的结果展示

## 📦 安装步骤

### 1. 加载扩展

1. 打开 Chrome 浏览器
2. 地址栏输入：`chrome://extensions/`
3. 右上角开启「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择 `boss-extension` 文件夹

### 2. 配置扩展

1. 点击浏览器右上角的扩展图标
2. 在弹出窗口中配置：
   - **关键词**：前端,JavaScript,React,Vue（逗号分隔）
   - **排除关键词**：外包,派遣,实习（逗号分隔）
   - **薪资范围**：15K - 40K
   - **城市代码**：101010100（北京）
   - **最大投递数**：50

### 3. 城市代码对照表

```
北京: 101010100
上海: 101020100
广州: 101280100
深圳: 101280600
杭州: 101210100
成都: 101270100
```

## 🚀 使用方法

### 方式一：手动搜索后投递（推荐）

1. 打开 BOSS直聘：https://www.zhipin.com
2. 手动搜索职位（输入关键词、选择城市等）
3. 在搜索结果页面，点击右下角的「🚀 开始自动投递」按钮
4. 等待自动投递完成

### 方式二：使用扩展配置

1. 在扩展弹窗中配置好关键词和城市
2. 点击「开始投递」
3. 扩展会自动跳转到搜索页面并开始投递

## ⚙️ 配置说明

### 关键词匹配

- 职位标题或描述包含任一关键词即可匹配
- 多个关键词用英文逗号分隔
- 示例：`前端,JavaScript,React,Vue,TypeScript`

### 排除关键词

- 包含任一排除关键词的职位会被跳过
- 示例：`外包,派遣,实习,兼职`

### 薪资范围

- 只投递薪资在此范围内的职位
- 单位：K（千元/月）

### 投递策略

- **最小延迟**：3秒（避免被检测）
- **最大延迟**：8秒（模拟人工）
- **批次大小**：每5个职位休息一次
- **批次间隔**：15秒

## 📊 匹配规则

扩展会根据以下规则计算职位匹配度（满分100分）：

- **关键词匹配**：40分（匹配越多分数越高）
- **薪资匹配**：30分（完全符合30分，部分符合15分）
- **经验匹配**：30分（符合要求30分）

**只有匹配度 ≥ 60分的职位才会投递**

## 🛡️ 安全特性

### 反检测机制

1. **随机延迟**：每次投递间隔3-8秒
2. **批次休息**：每投递5个休息15秒
3. **模拟人工**：滚动、鼠标悬停等行为
4. **页面限制**：只在搜索结果页注入脚本

### 防止重复投递

- 自动记录已投递的职位ID
- 下次运行时自动跳过已投递的职位

## 🔧 故障排查

### 问题1：页面不停刷新

**原因**：旧版本的bug已修复

**解决**：
1. 卸载旧扩展
2. 重新加载新版本
3. 刷新BOSS直聘页面

### 问题2：找不到职位卡片

**原因**：BOSS直聘更新了页面结构

**解决**：
1. 打开浏览器控制台（F12）
2. 查看是否有错误信息
3. 联系开发者更新选择器

### 问题3：投递失败

**原因**：可能触发了BOSS直聘的安全验证

**解决**：
1. 降低投递速度（增加延迟时间）
2. 减少每批投递数量
3. 手动完成验证后继续

### 问题4：没有找到"开始投递"按钮

**原因**：不在职位搜索页面

**解决**：
1. 确保URL包含 `/web/geek/job`
2. 手动搜索职位后再使用
3. 刷新页面重试

## 📝 使用建议

1. **首次使用**：建议设置较小的投递数量（如10个）测试
2. **关键词设置**：不要设置过于宽泛的关键词，避免投递不匹配的职位
3. **投递时间**：建议在工作日的工作时间投递，回复率更高
4. **每日限制**：建议每天投递不超过50个，避免被限制
5. **定期清理**：定期清理投递记录，避免数据过多

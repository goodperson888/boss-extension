# BOSS直聘自动投递助手 - Chrome扩展版

> 智能筛选 · 自动投递 · 高效求职

## 🎯 功能特点

- ✅ **智能筛选**: 根据关键词、经验等条件自动筛选职位
- ✅ **灵活匹配**: 自定义关键词匹配数量和排除规则
- ✅ **自动投递**: 一键批量投递符合条件的职位
- ✅ **个性化招呼语**: 自动生成并发送个性化招呼语
- ✅ **投递记录**: 自动记录投递历史，避免重复投递
- ✅ **数据导出**: 支持导出投递记录为CSV文件
- ✅ **防风控机制**: 随机延迟、模拟人工操作、批次休息
- ✅ **100%安全**: 运行在真实浏览器环境，不会被检测

## 📦 安装步骤

### 1. 加载扩展

1. 打开Chrome浏览器，访问 `chrome://extensions/`
2. 打开右上角的"开发者模式"开关
3. 点击"加载已解压的扩展程序"
4. 选择 `boss-extension` 文件夹
5. 扩展安装完成！

### 2. 配置扩展

1. 点击浏览器工具栏的扩展图标 🚀
2. 在弹出窗口中配置筛选条件：
   - **关键词**: 职位必须包含的关键词（如：前端,React,Vue）
   - **至少匹配几个关键词**: 控制匹配严格度（默认1个）
   - **排除关键词**: 要排除的关键词（如：外包,派遣,实习）
   - **包含几个排除关键词才跳过**: 控制排除严格度（默认1个）
   - **最大投递数量**: 一次最多投递多少个职位
   - **投递间隔**: 每次投递之间的延迟（建议2000-5000毫秒）
   - **招呼语模板**: 自定义招呼语，支持变量替换

3. 点击"保存配置"

## 🚀 使用方法

### 方式一：使用浮动按钮（推荐）

1. 登录 [BOSS直聘](https://www.zhipin.com)
2. 搜索你想要的职位（如：前端工程师）
3. 在页面右下角会出现"🚀 开始自动投递"按钮
4. 点击按钮，扩展会自动：
   - 分析页面上的所有职位
   - 自动滚动加载更多职位（懒加载）
   - 根据配置筛选符合条件的职位
   - 自动点击"立即沟通"
   - 发送个性化招呼语
   - 记录投递历史
5. 完成后会显示投递结果面板

### 方式二：使用扩展弹窗

1. 在BOSS直聘职位搜索页面
2. 点击浏览器工具栏的扩展图标
3. 点击"开始投递"按钮
4. 等待投递完成

### 控制按钮

- **⏸️ 暂停**: 临时暂停投递，可以继续
- **⏹️ 停止**: 彻底停止投递，清除进度

## 📊 查看投递记录

1. 点击扩展图标
2. 切换到"记录"标签
3. 查看所有投递历史
4. 可以导出为CSV文件进行分析

## ⚙️ 配置说明

### 关键词匹配规则

**关键词设置**：
- 支持多个关键词，用逗号分隔
- 职位标题或描述包含关键词即可匹配
- 示例：`前端,JavaScript,React,Vue`

**至少匹配几个关键词**：
- 默认：1个（宽松匹配）
- 设置为2：必须匹配至少2个关键词才投递（严格匹配）
- 示例：设置关键词为"前端,React,Vue"，匹配数为2，则必须同时包含"前端+React"或"前端+Vue"等至少2个才投递

### 排除关键词规则

**排除关键词设置**：
- 包含这些关键词的职位会被自动跳过
- 示例：`外包,派遣,实习,兼职`

**包含几个排除关键词才跳过**：
- 默认：1个（严格排除）
- 设置为2：必须包含至少2个排除关键词才跳过（宽松排除）
- 示例：设置排除词为"外包,派遣,实习"，排除数为2，则必须同时包含"外包+派遣"等至少2个才跳过

### 匹配度评分

- **关键词匹配**: 70分（匹配数量 ≥ 设置值）
- **经验匹配**: 30分（未设置或匹配时得分）
- **投递阈值**: ≥60分

### 招呼语变量

支持以下变量自动替换：
- `{experience}`: 职位要求的经验（如：3-5年）
- `{position}`: 职位名称
- `{company}`: 公司名称

示例模板：
```
您好，我对{company}的{position}职位很感兴趣，我有{experience}的相关经验，期待与您进一步沟通。
```

## 🛡️ 防风控机制

扩展内置多重防风控措施：

1. **随机延迟**：
   - 每次投递间隔：3-8秒随机
   - 点击延迟：200-500ms随机
   - 页面加载等待：800-1200ms随机

2. **批次休息**：
   - 每投递5个职位休息15秒
   - 避免频繁操作触发限制

3. **模拟人工操作**：
   - 鼠标悬停模拟
   - 平滑滚动动画
   - 随机等待时间

4. **智能滚动**：
   - 自动检测页面底部
   - 触发懒加载机制
   - 避免重复处理

## 💡 使用技巧

1. **首次使用建议**：
   - 先设置较小的投递数量（如10个）测试效果
   - 观察匹配度是否符合预期
   - 调整配置后再批量投递

2. **避免被限制**：
   - 设置合理的投递间隔（建议3000毫秒以上）
   - 不要一次投递过多职位（建议50个以内）
   - 分多次投递，每次间隔一段时间

3. **提高匹配度**：
   - 精确设置关键词和排除关键词
   - 合理设置匹配数量
   - 定期查看投递记录，优化配置

4. **灵活配置示例**：
   - **宽松匹配**：关键词5个，匹配数1个 → 只要包含任意1个即可
   - **严格匹配**：关键词3个，匹配数2个 → 必须包含至少2个
   - **精准匹配**：关键词2个，匹配数2个 → 必须全部包含

## 🐛 常见问题

### Q: 扩展没有反应？
A: 请刷新BOSS直聘页面后重试

### Q: 找不到浮动按钮？
A: 确保你在职位搜索结果页面，不是首页或其他页面

### Q: 投递失败？
A: 检查是否已登录BOSS直聘，或者页面结构是否有变化

### Q: 如何更新扩展？
A: 替换文件后，在 `chrome://extensions/` 页面点击扩展的刷新按钮

### Q: 日志面板太大/太小？
A: 日志面板支持拖动和调整大小，可以自由调整到合适的尺寸
